JavaScript 000 Basics

Javascript is a cross-platfor, object-oriented script language. JavaScript is extremely popular for a variety of reasons. It is small and lightweight language allowing maximum flexibiblity for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declaration
    JavaScript variable are containers for storing data values - imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identified with unique names. These unique names are called identifiers. var x;

  • Variable Assignment
    Assignmetn operators assign values to JavaScript variables - our cup can now have coffee poured in it, giving our variable a values to hold. The = assignment operator assigns a value to a variable. var x = 10;

  • Data Types
    Data Types are an important concept; to be able to operate on variables you need to know the data type. There are six data types that are JavaScript primitives: Bolean - true or false; null - null aka nother; Number - 42 or 3.14159; String - "Coding Dojo Rocks!"; Array - [1,'Coding',2,'Dojo']; and Object - {first_name: 'Jane', last_name: 'Doe'}

  • Conditionals
    When you write code, you want to perform different actions for different decisions - hittling different code blocks based on values or conditions that have been met. You can use condititional statemetns in your code to accomplis this. There are the following conditional statements: if a specified condition is true, do theis code in our code block; else if to specify a new codition to test, if the first condition is false; else we execute this block of code;

  • Loops
    There are many different kinds of loops in every programming language, but they all essential do the same thing: they will repeat an action some number of times. Imagine you have to run a mile, well you run around the track four times and then you top. Thats a loop!

  • Function
    Functions are an encaspulation of a code block. When we call our function this will run that code block. Think of it as a list of instructions. As an example imagine we are putting together a desk from Ikea, we open up the instruction manual and get started, first we screw the legs to the table top; next we place the table the right way up. Done! We finished our instructions. Sadly there are a ton more tables to do so lets call our function over and over and the over again.
